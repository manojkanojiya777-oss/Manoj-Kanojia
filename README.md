## Core Trading Rules
- Always trade in the direction of the trend
- Risk only 1-2% of capital per trade
- Avoid emotional or impulsive trades
- Follow setup confirmation strictly

## Current Trading Setup
- Timeframe: Daily
- Strategy: 200 EMA Pullback

### Entry Criteria
- Price above 200 EMA (uptrend)
- Pullback near EMA
- Bullish confirmation candle

### Exit Criteria
- Stop Loss below recent swing low
- Target based on risk-reward (minimum 1:2)
